# Play Store Asset Checklist

Required assets:
- App icon: 512 x 512 PNG
- Feature graphic: 1024 x 500 PNG
- Phone screenshots: at least 2 (1080 x 1920 recommended)
- 7-inch and 10-inch tablet screenshots (optional but recommended)
- Short description (80 chars)
- Full description (4000 chars)
- Privacy policy URL

This app project includes release builds and a multi-screen UI for store screenshots.
